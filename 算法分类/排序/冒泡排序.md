### 冒泡排序

* 时间复杂度 O(n^2)
* 空间复杂度 O(1)


```js
function sort(nums) {
    if (Array.isArray(nums)) {
        for (let i = 0; i < nums.length; i++) {
            for (let j = i + 1; j < nums.length; j++) {
                if (nums[i] > nums[j]) {
                    let temp = nums[i];
                    nums[i] = nums[j];
                    nums[j] = temp;
                }
            }
        }
        return nums;
    }
    return []
}
```
