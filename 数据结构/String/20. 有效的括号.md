给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。


### 思路

遍历字符串,将扩号分成左括号和右括号
左右括号的长度一致 那么它是符合规则的


```js
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
    if (s === null) return false;
    if (!s.length) return false;
    
    
};
```
