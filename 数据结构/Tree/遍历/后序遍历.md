同理，有了前面的分析，后续就是左子树 ---> 右子树 ---> 根结点

```js
function dfs (root) {
    if (root !== null) {
        dfs(root.left);
        dfs(root.right);
        console.log(root.val);
    }
}
```


### 非递归后序

非递归后序遍历有两种方法

* 一种方法是利用和前面中序、前序第二种方法类似的方法进入压栈出栈，但是要借助额外的标记次数，一个节点访问第二次才能输出。
* (这个访问第一次是入栈，第二次是子树解决完毕自己即将出栈（先不出栈）)。
